# Slay the Spire 2 Trainer (Рабочий чит / Working Cheats)

**English**  
Full-featured external trainer for **Slay the Spire 2** (Early Access, release March 05, 2026).  
Tested and working on the current Steam version.

**Русский**  
Полноценный внешний трейнер для **Slay the Spire 2** (Early Access, релиз 05.03.2026).  
Протестировано и работает на актуальной версии в Steam.

**Features / Функции (все работают):**
- Infinite Energy (бесконечная энергия каждый ход)
- God Mode / Infinite HP (неуязвимость / бесконечное здоровье)
- Infinite / Mega Gold (бесконечное золото, редактирование)
- Unlimited Block (бесконечный блок)
- Zero Cost Cards (карты без стоимости)
- One Hit Kill Enemies (убийство с одного удара)
- Instant Win Combat (мгновенная победа в бою)
- Unlock All Cards & Relics (разблокировка всех карт и реликвий)
- Infinite Potions (бесконечные зелья)
- Edit Gold / HP / Deck Size (редактирование золота, HP, размера колоды)
- No RNG / Fixed Seeds (отключение случайности / фиксированные сиды)
- Super Fast Animations (ускоренная анимация)
- Reveal Map (открытие всей карты)
- и другие (~15–20 опций)

**How to use / Как использовать:**
1. Launch the game (Slay the Spire 2 must be running)  
   Запусти игру (Slay the Spire 2 должна быть открыта)
2. Run the trainer as administrator (`STS2Trainer.exe`)  
   Запусти трейнер от имени администратора
3. Activate desired cheats using hotkeys (Num 1–9, F1–F10, etc.)  
   Активируй нужные читы горячими клавишами
4. Enjoy — everything works without crashes  
   Наслаждайся — всё работает без вылетов

**Antivirus warning / Предупреждение антивируса**  
Most antiviruses flag trainers — add an exception.  
Большинство антивирусов ругаются на читы — добавь исключение.

**Download / Скачать**  
→ Go to **Releases** → latest version → download `STS2-Trainer-v1.0.zip`  
→ Перейди во вкладку **Releases** → последняя версия → скачай `STS2-Trainer-v1.0.zip`

**Build from source / Сборка из исходников**  
- Visual Studio 2022+ (C++ Desktop workload)  
- Open `.sln` or create project from `src/`  
- Build → Release → x64  
- Executable will be in `x64/Release/` or `Release/`
